# Personal Portfolio — Chukwunomso

A modern, dark-themed developer portfolio built with React, Vite and Tailwind CSS.

## Sections
- Hero — introduction and links
- About — background, stack and resume
- Projects — live projects with links
- Skills — tech stack and skill levels
- Contact — social links and email

## Built With
- React
- Tailwind CSS
- Vite
- JavaScript
